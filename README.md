Sistema web desenvolvido em PHP com integração ao banco de dados MySQL para gestão de produtos. Implementa as quatro operações básicas CRUD — Create, Read, Update e Delete — permitindo cadastrar, listar, atualizar e eliminar produtos.
Tecnologias

PHP

MySQL

HTML

CSS

Funcionalidades

Cadastro de produtos

Listagem de produtos

Edição de informações

Eliminação com confirmação

Interface simples e funcional
